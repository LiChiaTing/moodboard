# Wireframe changelog

A short note on what changed between wireframe versions. The latest version lives
at `wireframe-desktop.html`; older versions are kept in `iterations/`.

| Version | File | What changed |
|---------|------|--------------|
| Latest | `wireframe-desktop.html` | Desktop wireframe the React app was built from |
| v2 | `iterations/wireframe-v2.html` | Earlier full-flow iteration |
| Early | `iterations/moodboard-wireframe.html` | First Moodboard wireframe |
| Review | `iterations/moodboard-wireframe-review.html` | Review/markup pass on the wireframe |

> When a wireframe evolves: copy the current file into `iterations/` first (to keep
> the old version visible), then edit the latest, and add a row here describing the change.
