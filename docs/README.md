# Docs

This folder holds the thinking behind Moodboard: research, design, and the
decisions made along the way. Folders are numbered so they read in workflow order.

## 01-research

User research that informed the design.

- `moonboard-ux-research.html` — UX research (English)
- `moonboard-ux-research-zh.html` — UX research (中文)

## 02-design

The design work, from system to screens.

- `design-system/` — the visual language
  - `DESIGN_SYSTEM.md` / `design-system.html` — colours, type, spacing, components
  - `logo/` — logo source files (`logo.svg`, `logo.html`, `logo-design.html`)
  - `references/` — reference / inspiration images
- `ui/` — interface design
  - `wireframe-desktop.html` — latest wireframe (the app was built from this)
  - `moodboard-refinement-spec.html` — UI refinement spec
  - `iterations/` — earlier wireframe versions, kept for history
- `SCREEN_DATA_NEEDS.md` — what data each screen needs

## 03-decisions

- `decision-log.md` — key design decisions and the reasoning behind them
  (also the raw material for case studies)

## File formats

- `.html` — the living, editable working version (open in a browser)
- `.pdf` — a frozen snapshot for sharing (export from the HTML)
- `.md` — lightweight notes and decisions
